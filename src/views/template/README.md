# Template 模块目录说明

本文档用于记录 `src/views/template` 下各业务目录的目的与功能。

## 目录功能

### `customer`

- 目的：客户管理
- 功能：销售管理所需要的数据源

### `dictionary`

- 目的：字典管理
- 功能：一些标准字段的自定义数据存储，支持增删改查

### `item-dict`

- 目的：物料字典
- 功能：维护品类/规格/品牌树及编码，供物料编码自动生成

### `items`

- 目的：物料管理
- 功能：物料分成品和零件两种；品类/规格/品牌来自物料字典

### `supplier`

- 目的：供应商管理

### `unit`

- 目的：单位管理
- 功能：用于特殊单位与单位换算

### `warehouse`

- 目的：仓库管理
